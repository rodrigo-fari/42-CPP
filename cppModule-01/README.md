# C++ Module 01

## Overview

This module deepens core C++ concepts through practical exercises.
It typically covers memory allocation, references, pointers, and object lifecycle basics.

## Repository Layout

- `ex00/`
- `ex01/`
- `ex02/`
- `ex03/`
- `ex04/`
- `ex05/`

Each exercise is isolated and includes its own `Makefile`.

## Build

From inside an exercise folder:

```bash
make
```

## Run

After building, execute the generated binary from that exercise directory.

## Notes

- Respect the expected output and behavior from the subject PDF.
- Prefer clear, readable class design and ownership rules.
- Validate with both normal and edge-case inputs.
