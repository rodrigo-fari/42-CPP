# C++ Module 02

## Overview

This module focuses on canonical class form and operator overloading in C++.
The exercises reinforce proper class design, constructors, copy semantics, and arithmetic/operator behavior.

## Repository Layout

- `ex00/`
- `ex01/`
- `ex02/`

Each exercise is independent and includes its own `Makefile`.

## Build

From inside an exercise folder:

```bash
make
```

## Run

Run the executable generated by `make` in that directory.

## Notes

- Ensure copy constructor and assignment operator behavior is correct.
- Keep operator overloads consistent and predictable.
- Verify numerical logic with representative test cases.
