# C++ Module 05

## Overview

This module introduces exceptions and robust error handling patterns in C++.
It focuses on safe class behavior, validation rules, and clear failure management.

## Repository Layout

- `ex00/`
- `ex01/`
- `ex02/`
- `ex03/`

Each exercise directory contains standalone source code and a dedicated `Makefile`.

## Build

From inside an exercise folder:

```bash
make
```

## Run

Execute the produced binary from the corresponding folder.

## Notes

- Throw exceptions for invalid states as required by the subject.
- Catch exceptions at appropriate boundaries with meaningful handling.
- Keep error paths testable and deterministic.
