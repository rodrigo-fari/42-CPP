# C++ Module 09

## Overview

This module consolidates previous concepts through larger algorithmic exercises.
It emphasizes data structures, parsing, and implementation correctness under constraints.

## Repository Layout

- `ex00/`
- `ex01/`
- `ex02/`

Exercises are split into independent folders, each with its own `Makefile`.

## Build

From inside an exercise folder:

```bash
make
```

## Run

Execute the generated binary in the same exercise directory.

## Notes

- Prioritize correctness before optimization.
- Keep parsing and validation logic explicit and testable.
- Add representative test scenarios for expected and invalid input.
