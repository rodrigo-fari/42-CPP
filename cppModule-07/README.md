<h3 align="center">Overview</h3>

This module introduces templates for generic programming in C++.
It focuses on reusable type-agnostic utilities and compile-time polymorphism.

---

<h3 align="center">Repository Layout</h3>

- `ex00/`
- `ex01/`
- `ex02/`

Every exercise folder is independent and contains its own `Makefile`.

---

<h3 align="center">Build</h3>

From inside an exercise folder:

```bash
make
```

---

<h3 align="center">Run</h3>

Execute the generated binary from the corresponding exercise directory.

---

<h3 align="center">Notes</h3>

- Keep template interfaces minimal and clear.
- Prefer generic code over duplicated type-specific logic.
- Test with multiple data types to verify correctness.
