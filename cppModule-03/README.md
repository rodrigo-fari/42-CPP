# C++ Module 03

## Overview

This module introduces inheritance and polymorphic behavior through game-like class hierarchies.
It emphasizes constructor chaining, access control, and extending base-class functionality.

## Repository Layout

- `ex00/`
- `ex01/`
- `ex02/`

Each exercise is self-contained and provides its own `Makefile`.

## Build

From inside an exercise folder:

```bash
make
```

## Run

Execute the binary produced by the build process.

## Notes

- Keep inheritance relationships explicit and coherent.
- Reuse base functionality instead of duplicating logic.
- Validate behavior for both base and derived class objects.
