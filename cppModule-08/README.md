# C++ Module 08

## Overview

This module focuses on STL containers, iterators, and algorithm usage.
It develops practical skills for working with generic collections efficiently.

## Repository Layout

- `ex00/`
- `ex01/`
- `ex02/`

Each exercise provides standalone implementation files and a dedicated `Makefile`.

## Build

From inside an exercise folder:

```bash
make
```

## Run

Run the compiled program from the exercise directory.

## Notes

- Choose containers according to access and performance needs.
- Use iterator-safe patterns and avoid invalidation pitfalls.
- Validate algorithm behavior on small and edge-case datasets.
