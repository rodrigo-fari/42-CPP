# C++ Module 00

## Overview

This module introduces the fundamentals of C++ in the 42 curriculum.
It focuses on basic syntax, classes, member functions, and simple program structure.

## Repository Layout

- `ex00/`
- `ex01/`

Each exercise is self-contained and includes its own `Makefile`.

## Build

From inside an exercise folder:

```bash
make
```

## Run

After building, run the generated executable from the same folder.

## Notes

- Follow the subject requirements for each exercise.
- Keep compilation flags strict (as defined in the provided Makefiles).
- Test edge cases and invalid input when relevant.
